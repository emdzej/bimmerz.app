# ncsx — user guide

> This guide is a stub. Content is coming as the docs settle.

Things this guide will cover:

- Picking your BMW Standard Tools install + connecting the cable.
- Reading vehicle identity (VIN + FA or ZCS) and what each field means.
- Browsing the chassis ECU catalogue and what gets filtered by FA.
- Reading current coding + interpreting the FSW / PSW tree.
- Editing a value safely — staging, byte diff, and confirming the write.
- Editing FA tokens / ZCS bits in the modal dialogs.
- Applying factory defaults (when supported by the CABD).
- Running ad-hoc IPO jobs from the "Other jobs" dropdown.
- Exporting / importing `FSW_PSW.TRC`, `FSW_PSW.MAN`, `NETTODAT.TRC`.

In the meantime, the [project README](https://github.com/emdzej/ncsx)
walks the basic read → edit → write flow.
