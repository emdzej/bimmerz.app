# TISX — user guide

> This guide is a stub. Content is coming as the docs settle.

Things this guide will cover:

- Selecting your vehicle by series / model / engine, or by VIN.
- Browsing the document tree (document type → main group → sub group).
- Reading a repair procedure — hotspot cross-references, image embedding,
  print layout.
- Symptom-based navigation through the complaint tree.
- Bookmarking vehicles + documents to favourites, exporting / importing.
- Toggling dark mode.

In the meantime, the [project README](https://github.com/emdzej/tisx) has
the data-preparation walkthrough.
